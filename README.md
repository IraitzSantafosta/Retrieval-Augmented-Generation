# Retrieval-Augmented-Generation
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/IraitzSantafosta/Retrieval-Augmented-Generation/blob/main/Retrieval_Augmented_Generation.ipynb)
Este proyecto presenta el desarrollo de un sistema RAG especializado en medicina diseñado para extraer respuestas precisas y fundamentadas de documentos técnicos. El proceso comienza con la recopilación de datos de fuentes como PubMedQA y MedExpQA, que aportan un contexto científico y natural para las consultas. Para la fase de recuperación de información, se emplean herramientas como BioBERT y FAISS, las cuales permiten localizar los fragmentos de texto más pertinentes de forma eficiente. Posteriormente, la generación de respuestas se lleva a cabo mediante modelos de lenguaje avanzados como T5 y SciFive, este último optimizado para la terminología biomédica. Finalmente, la calidad y veracidad de los resultados se miden utilizando métricas de evaluación estándar como ROUGE, BLEU y METEOR. El objetivo principal es mitigar los riesgos de información errónea en el ámbito sanitario proporcionando respuestas razonadas y basadas en evidencia.
